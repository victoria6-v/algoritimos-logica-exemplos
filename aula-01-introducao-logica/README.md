# Aula 01 — Introdução à Lógica de Programação

## 📋 Conteúdo da Aula

Nesta aula foram apresentados os fundamentos da lógica de programação:

- O que é lógica e por que ela é importante na programação
- O que é um algoritmo (sequência de passos para resolver um problema)
- O modelo **Entrada → Processamento → Saída**
- Exemplos do cotidiano transformados em algoritmos
- Conceito de variáveis e tipos de dados
