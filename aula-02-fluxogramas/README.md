# Aula 02 — Fluxogramas, Variáveis e Operadores

## 📋 Conteúdo da Aula

- Símbolos do fluxograma (início/fim, processo, decisão, entrada/saída)
- Declaração de variáveis e tipos de dados: inteiro, real, literal (string)
- Operadores aritméticos: `+`, `-`, `*`, `/`
- Modelo **Entrada → Processamento → Saída** aplicado na prática

## 🛠️ Ferramentas Utilizadas

- **Draw\.io** — Criação dos fluxogramas visuais
- **VisualG** — Pseudocódigo em português
